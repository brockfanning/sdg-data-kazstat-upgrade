Kazstat SDG data repository - Open SDG output

This output includes a variety of endpoints. Examples are below:

Headlines

CSV/JSON files containing "headline" (aggregated only) data for indicators.

  • ru/headline/1-1-1.csv
  • ru/headline/1-1-1.json
  • kk/headline/1-1-1.csv
  • kk/headline/1-1-1.json
  • en/headline/1-1-1.csv
  • en/headline/1-1-1.json

Data

CSV/JSON files containing fully disaggregated data for indicators.

  • ru/data/1-1-1.csv
  • ru/data/1-1-1.json
  • kk/data/1-1-1.csv
  • kk/data/1-1-1.json
  • en/data/1-1-1.csv
  • en/data/1-1-1.json

Edges

CSV/JSON files containing "edges" (relationships between disaggregations) data for indicators.

  • ru/edges/1-1-1.csv
  • ru/edges/1-1-1.json
  • kk/edges/1-1-1.csv
  • kk/edges/1-1-1.json
  • en/edges/1-1-1.csv
  • en/edges/1-1-1.json

Combined edges and data

JSON files containing both the fully-disaggregated and "edges" data mentioned above.

  • ru/comb/1-1-1.json
  • kk/comb/1-1-1.json
  • en/comb/1-1-1.json

Metadata

JSON files containing metadata for the indicators.

  • ru/comb/1-1-1.json
  • kk/comb/1-1-1.json
  • en/comb/1-1-1.json

Zip file of CSV data

Zip files containing all indicators in CSV form.

  • ru/zip/all_indicators.zip
  • kk/zip/all_indicators.zip
  • en/zip/all_indicators.zip

Zip file information

JSON file containing information about the above-mentioned zip files.

  • ru/zip/all_indicators.json
  • kk/zip/all_indicators.json
  • en/zip/all_indicators.json

Reporting status

JSON file containing information about the reporting status of the indicators.

  • ru/stats/reporting.json
  • kk/stats/reporting.json
  • en/stats/reporting.json

Translations

JSON file containing all the translations used in the platform.

  • ru/translations/translations.json
  • kk/translations/translations.json
  • en/translations/translations.json
